## 特點 Features
+ 將每一次的運動記錄起來
+ 使用者可以自訂運動類型
+ 透過月曆上每日不同的顏色深淺一目了然這個月的運動量
+ 一年內、一周間的卡路里表
+ 透過養成遊戲、為了餵食獅子更努力的運動吧!

## 技術/環境
+ 使用 Visual Studio 2015
+ 主要語言為 C#
+ 串接mySQL作為資料庫

## 優化UI
### 紀錄下載時機
+ 每一次只吃進幾筆資料
+ 減少使用者等待時間
+ 根據卷軸位置決定吃進資料的時機
